## Työn aihe

Työn aiheena on toteuttaa verkkokauppa. Ylläpitäjät voivat hallita verkkokaupan tuotteita, ja asiakkaat voivat tilata tuotteita. Tuotteita voidaan hakea esimerkiksi kategorian perusteella.
